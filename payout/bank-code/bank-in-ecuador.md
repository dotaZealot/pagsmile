## Account Number

| Bank                                        | Description                                                                                      |
| ------------------------------------------- | ------------------------------------------------------------------------------------------------ |
| Banco Pichincha C.A.                        | The “No. account ”must contain 10 digits                                                         |
| For any other bank or financial institution | A 20-digit CCI interbank account code must be entered. Example: 00219400254640654321 (20 digits) |
